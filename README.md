# Auth
---
An example of an authentication system with vue.js, node.js and jwt.

## Installation


Install the package.

**1.Server**

```bash
npm install 
```

**2.Frontend**

```bash
cd frontend
npm install 
```

**3.Rename the .env.example file to .env and insert your values.**


## And start the program
**Server**
```bash
nodemon serve
```

**Frontend**
```bash
cd frontend
npm run serve
```
